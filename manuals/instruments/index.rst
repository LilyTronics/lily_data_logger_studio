Supported instruments
=====================

The table below lists all supported instrumnets.

============ ======================= =======================================================
Manufacturer Model                   Description
============ ======================= =======================================================
Aim-TTi      PL303                   Aim-TTi power supply PL303, dual channel 15V/3A
Arduino      DAQ                     Basic DAQ using an Arduino board
Tektronix    TDS1000 (RS232)         Tektronix oscilloscope TDS1000 series using RS232
\            TDS200(RS232)           Tektronix oscilloscope TDS200 series using RS232
\            TDS2000 (RS232)         Tektronix oscilloscope TDS2000 series using RS232
Voltcraft    VC506                   Multimeter Voltcraft VC506
Simulator    multi channel analog IO Simulates a multi channel analog IO device with ASCII

                                     protocol over IP/TCP
\            temperature chamber     Simulates a temperature chamber with ASCII protocol

                                     over IP/UDP
