Lily Data Logger Studio Instruments Package
===========================================

This document lists all instruments that are supported by Lily Data Logger Studio.

Instruments can easily be added to the instruments package by creating a new driver.
See the manual 'Driver development' for more information.

You can also send us a request for support for a particular instrument.
See our website for contact details: https://lilytronics.nl
