Drivers
-------

When the application starts, it load the driver from the instrumens package.
Depending on the amount of drivers this can take some time.
A progress window is shown when drivers are loaded.

It is possible to create your own drivers for your own instruments.
A separate manual is available for that.

When a driver is added, the drivers can be reloaded with the following toolbar button:

.. image:: images/drivers_reload_button.png

Reloading drivers can affect measurements when the data logger is running.
Stop the data logger first before reloading the drivers.
