Downloading and installation
----------------------------

The software can be downloaded from the `release pages`_ on github. Download the zip file
that is appropriate for your operating system. After downloading, unpack the zip file into an
empty folder (e.g.: C:\\LilyDataLoggerStudio). This can be any folder. After unpacking you will
find the following items:

* instruments: this folder contains the drivers for the supported electronic instruments.
* lib: this folder contains all libraries that are required for executing the software.
* manuals: this folder contains, this manual and a manual for developing drivers for your instruments.
* LilyDataLoggerStudio(.exe): this is the executable for the main application.
* LilyDataLoggerStudioDriverTest(.exe): this is the executable for the driver test application

For normal operation you only need to execute the ``LilyDataLoggerStudio`` executable.
